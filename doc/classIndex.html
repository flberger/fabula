Class Hierarchy

  • fabula.assets.Assets - An assets manager which returns file-like objects for local files.
  • fabula.Event - Fabula event base class.
    • fabula.AttemptEvent - This is the base class for attempt events by the player or NPCs.
      • fabula.TriesToDropEvent - The player or NPC tries to drop an item.
      • fabula.TriesToLookAtEvent - Issued when the player or an NPC looks at a map element or an item.
      • fabula.TriesToManipulateEvent - Issued when the player or an NPC tries to manipulate an item.
      • fabula.TriesToMoveEvent - Issued when the player or an NPC wants to move, awaiting server confirmation.
      • fabula.TriesToPickUpEvent - Issued when the player or an NPC tries to pick up an item.
      • fabula.TriesToTalkToEvent - Issued when the player or an NPC wants to start a conversation with another entity.
    • fabula.ChangePropertyEvent - This Event is issued to trigger a property change.
    • fabula.ConfirmEvent - This is the base class for server confirmations of attempt events.
      • fabula.AttemptFailedEvent - Server announcement that the last AttemptEvent failed.
      • fabula.CanSpeakEvent - This is a server invitation for the player or an NPC to speak.
      • fabula.DropsEvent - This is the server confirmation for an attempt to drop an item
      • fabula.ManipulatesEvent - This is a server confirmation of an item manipulation.
      • fabula.MovesToEvent - This is the server confirmation of a movement.
      • fabula.PerceptionEvent - This is a perception for the player or an NPC issued by the server,
      • fabula.PicksUpEvent - This is a server confirmation of a TriesToPickUpEvent
    • fabula.ExitEvent - This Event is sent by the Client or the Server to signal that it is about to exit and that no further Events will be sent.
    • fabula.InitEvent - This event is sent by the client upon startup.
    • fabula.PassiveEvent - The base class for events when items or NPCs are being passed etc.
      • fabula.DroppedEvent - Issued by the server when an item is being dropped by the player or an NPC.
      • fabula.LookedAtEvent - Issued by the server when an item or NPC is being looked at by the player or an NPC.
      • fabula.PassedEvent - Issued by the server when an item or an NPC is being passed by the player or an NPC.
      • fabula.PickedUpEvent - Issued by the server when an item is being picked up by the player or an NPC.
    • fabula.SaysEvent - This normally is a reaction to the CanSpeakEvent, to be sent by the player.
    • fabula.ServerEvent - Base class for various events issued by the server
      • fabula.ChangeMapElementEvent - This event changes a single element of the two-dimensional map.
      • fabula.DeleteEvent - This event deletes an item or NPC from the map.
      • fabula.EnterRoomEvent - This event announces that the player enters a new room.
      • fabula.RoomCompleteEvent - Issued by the server after an EnterRoomEvent when the room is populated with a tiles, a player Entity, items and NPCs.
      • fabula.ServerParametersEvent - This Event is sent by the Server when an InitEvent has been received.
      • fabula.SpawnEvent - This event creates the player / new item / NPC on the map.
  • fabula.eventprocessor.EventProcessor - This is the base class for all Fabula objects that process events.
    • fabula.core.Engine - Common base class for Fabula server and client engines.
      • fabula.core.client.Client - An instance of this class is the main engine in every Fabula client.
      • fabula.core.server.Server - The Server is the central management and control authority in Fabula.
    • fabula.plugins.Plugin - Base class for plugins to be used by a fabula.core.Engine.
      • fabula.plugins.serverside.DefaultGame - This is an off-the-shelf server plugin, running a default Fabula game.
        • fabula.plugins.serverside.CommandLine - This is a command-line interface server plugin.
        • fabula.plugins.serverside.Editor - This is the server-side Plugin for a Fabula map editor.
      • fabula.plugins.ui.UserInterface - This is the base class for an UserInterface for the Fabula Client.
        • fabula.plugins.pygameui.PygameUserInterface - This is a Pygame implementation of an UserInterface for the Fabula Client.
          • fabula.plugins.pygameui.PygameEditor - A Pygame-based user interface that serves as a map editor.
  • fabula.FloorPlanElement - Convenience class for floor plan elements.
  • fabula.interfaces.Interface - This is a base class for Fabula interfaces which handle all the network traffic.
    • fabula.interfaces.python_tcp.TCPClientInterface - Fabula Client interface using TCP.
    • fabula.interfaces.python_tcp.TCPServerInterface - Fabula Server interface using TCP.
      • fabula.interfaces.json_rpc.JSONRPCServerInterface - Fabula Server interface using JSON-RPC.
    • fabula.interfaces.replay.PythonReplayInterface - An Interface which replays Python message logs.
    • fabula.interfaces.StandaloneInterface - An Interface that is meant to be used in conjunction with run.App.run_standalone().
  • fabula.interfaces.MessageBuffer - Buffer messages received and to be sent over the network.
  • fabula.Message - A Message represents a set of Events that happen in parallel.
  • fabula.plugins.pygameui.planes.gui.Container
    • fabula.plugins.pygameui.EventEditor - A planes Container that implements an editor for Fabula Events.
  • fabula.plugins.pygameui.planes.gui.Fonts - A font manager.
  • fabula.plugins.pygameui.planes.gui.lmr.LMRStyle - This class encapsulates the left, mid and right images to be used as widget background, as well as the text color.
  • fabula.plugins.pygameui.planes.gui.lmr.LMRWidget - Base class for fixed-height, flexible-width widgets with an LMR background.
    • fabula.plugins.pygameui.planes.gui.lmr.LMRButton - A planes.gui.Button with LMR background.
    • fabula.plugins.pygameui.planes.gui.lmr.LMROption - A planes.gui.Option with LMR background.
  • fabula.plugins.pygameui.planes.gui.tmb.TMBStyle - This class encapsulates the top, mid and bottom images to be used as widget background.
  • fabula.plugins.pygameui.planes.Plane - A Plane is a surface in a hierarchy of surfaces.
    • fabula.plugins.pygameui.EntityPlane - Subclass of Plane with an extended update() method that handles movement.
    • fabula.plugins.pygameui.planes.Display - planes main screen class.
  • fabula.plugins.pygameui.planes.Stats - A Stats instance stores and computes several runtime statistics.
  • fabula.plugins.pygameui.PygameOSD - A PygameOSD instance manages an on-screen display.
  • fabula.plugins.pygameui.surfacecatcher.SurfaceCatcher - A SurfaceCatcher instance exposes an API to catch screen dumps.
  • fabula.Rack - A Rack stores Entities removed from a room.
  • fabula.run.App - An App instance represents a Fabula client or server application.
  • fabula.Tile - A Tile is an element of a two-dimensional map.
  • Interface
    • fabula.interfaces.python_udp.UDPClientInterface - This is the base class for a Fabula Client Interface.
    • fabula.interfaces.python_udp.UDPServerInterface - This is the base class for a Fabula Server Interface.
    • fabula.interfaces.twisted_tcp.TCPInterface - A generic Fabula Interface using TCP, built upon the Twisted framework.
  • MessageBuffer
    • fabula.interfaces.python_udp.UDPClientInterface - This is the base class for a Fabula Client Interface.
    • fabula.interfaces.twisted_tcp.ProtocolMessageBuffer - Buffer messages received and to be sent over the network.
  • planes.gui.Button
    • fabula.plugins.pygameui.planes.gui.lmr.LMRButton - A planes.gui.Button with LMR background.
  • planes.gui.Container
    • fabula.plugins.pygameui.planes.gui.tmb.TMBContainer - A planes.gui.Container with fixed width and TMB background.
      • fabula.plugins.pygameui.planes.gui.tmb.TMBFadingContainer - A planes.gui.FadingContainer with fixed width and TMB background.
      • fabula.plugins.pygameui.planes.gui.tmb.TMBGetStringDialog - A combination of TMBContainer, Label, TextBox and Button that asks the user for a string.
      • fabula.plugins.pygameui.planes.gui.tmb.TMBOkBox - A box which displays a message and an LMR OK button over a TMB background.
      • fabula.plugins.pygameui.planes.gui.tmb.TMBOptionSelector - A TMBOptionSelector wraps an lmr.LMROptionList and an OK button over a TMB background, calling a callback when a selection is confirmed.
  • planes.gui.FadingContainer
    • fabula.plugins.pygameui.planes.gui.tmb.TMBFadingContainer - A planes.gui.FadingContainer with fixed width and TMB background.
  • planes.gui.GetStringDialog
    • fabula.plugins.pygameui.planes.gui.tmb.TMBGetStringDialog - A combination of TMBContainer, Label, TextBox and Button that asks the user for a string.
  • planes.gui.OkBox
    • fabula.plugins.pygameui.planes.gui.tmb.TMBOkBox - A box which displays a message and an LMR OK button over a TMB background.
  • planes.gui.Option
    • fabula.plugins.pygameui.planes.gui.lmr.LMROption - A planes.gui.Option with LMR background.
  • planes.gui.OptionList
    • fabula.plugins.pygameui.planes.gui.lmr.LMROptionList - A planes.gui.OptionList with LMROption elements.
  • planes.gui.OptionSelector
    • fabula.plugins.pygameui.planes.gui.tmb.TMBOptionSelector - A TMBOptionSelector wraps an lmr.LMROptionList and an OK button over a TMB background, calling a callback when a selection is confirmed.
  • planes.gui.PlusMinusBox
    • fabula.plugins.pygameui.planes.gui.lmr.LMRPlusMinusBox - A planes.gui.PlusMinusBox with LMRButtons.
  • planes.Plane
    • fabula.plugins.pygameui.planes.gui.Container - A Container for Planes.
      • fabula.plugins.pygameui.planes.gui.FadingContainer - A Container that, once visible, will fade out and destroy itself.
      • fabula.plugins.pygameui.planes.gui.GetStringDialog - A combination of Container, Label, TextBox and Button that asks the user for a string.
      • fabula.plugins.pygameui.planes.gui.OkBox - A box which displays a message and an OK button.
      • fabula.plugins.pygameui.planes.gui.OptionList - A list of options to select from.
      • fabula.plugins.pygameui.planes.gui.OptionSelector - An OptionSelector wraps an OptionList and an OK button, calling a callback when a selection is confirmed.
    • fabula.plugins.pygameui.planes.gui.Label - A planes.Plane which displays a text.
      • fabula.plugins.pygameui.planes.gui.Button - A planes plane which displays a text and reacts on mouse clicks.
      • fabula.plugins.pygameui.planes.gui.Option - A subclass of Label which handles mouseclicks, to be used in an OptionList.
      • fabula.plugins.pygameui.planes.gui.OutlinedText - A Label with outlined text and a transparent background.
      • fabula.plugins.pygameui.planes.gui.TextBox - A box where the user can type text.
    • fabula.plugins.pygameui.planes.gui.PlusMinusBox - This class implements a TextBox with plus and minus buttons attached, to change a numerical value.
    • fabula.plugins.pygameui.planes.gui.ScrollingPlane - This class implements a fixed-dimension plane with a scroll bar to scroll its content plane.
  • twisted.internet.protocol.Protocol
    • fabula.interfaces.twisted_tcp.ProtocolMessageBuffer - Buffer messages received and to be sent over the network.
