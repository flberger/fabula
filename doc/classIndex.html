Class Hierarchy

  • clickndrag.Plane
    • shard.plugins.pygameui.EntityPlane - Subclass of Plane with an update() method that handles movement.
  • logging.Formatter
    • shard.run.ShardFileFormatter - Subclass of Formatter with reasonable module information.
  • shard.assets.Assets - An assets manager which returns file-like objects for local files.
  • shard.Event - Shard event base class.
    • shard.AttemptEvent - This is the base class for attempt events by the player or NPCs.
      • shard.TriesToDropEvent - The player or NPC tries to drop an item.
      • shard.TriesToLookAtEvent - Issued when the player or an NPC looks at a map element or an item.
      • shard.TriesToManipulateEvent - Issued when the player or an NPC tries to manipulate an item.
      • shard.TriesToMoveEvent - Issued when the player or an NPC wants to move, awaiting server confirmation.
      • shard.TriesToPickUpEvent - Issued when the player or an NPC tries to pick up an item.
      • shard.TriesToTalkToEvent - Issued when the player or an NPC wants to start a conversation with another entity.
    • shard.ChangeStateEvent - This Event is issued to trigger a state change.
    • shard.ConfirmEvent - This is the base class for server confirmations of attempt events.
      • shard.AttemptFailedEvent - Server announcement that the last AttemptEvent failed.
      • shard.CanSpeakEvent - This is a server invitation for the player or an NPC to speak.
      • shard.DropsEvent - This is the server confirmation for an attempt to drop an item
      • shard.ManipulatesEvent - This is a server confirmation of an item manipulation.
      • shard.MovesToEvent - This is the server confirmation of a movement.
      • shard.PerceptionEvent - This is a perception for the player or an NPC issued by the server,
      • shard.PicksUpEvent - This is a server confirmation of a TriesToPickUpEvent
    • shard.InitEvent - This event is sent once by the client upon startup.
    • shard.PassiveEvent - The base class for events when items or NPCs are being passed etc.
      • shard.DroppedEvent - Issued by the server when an item is being dropped by the player or an NPC.
      • shard.LookedAtEvent - Issued by the server when an item or NPC is being looked at by the player or an NPC.
      • shard.PassedEvent - Issued by the server when an item or an NPC is being passed by the player or an NPC.
      • shard.PickedUpEvent - Issued by the server when an item is being picked up by the player or an NPC.
    • shard.SaysEvent - This normally is a reaction to the CanSpeakEvent, to be sent by the player.
    • shard.ServerEvent - Base class for various events issued by the server
      • shard.ChangeMapElementEvent - This event changes a single element of the two-dimensional map.
      • shard.DeleteEvent - This event deletes an item or NPC from the map.
      • shard.EnterRoomEvent - This event announces that the player enters a new room.
      • shard.RoomCompleteEvent - Issued by the server after an EnterRoomEvent
      • shard.SpawnEvent - This event creates the player / new item / NPC on the map.
  • shard.eventprocessor.EventProcessor - This is the base class for all Shard objects that process events.
    • shard.core.Engine - Common base class for Shard server and client engines.
      • shard.core.client.Client - An instance of this class is the main engine in every Shard client.
      • shard.core.server.Server - The Server is the central management and control authority in Shard.
    • shard.plugins.Plugin - Base class for plugins to be used by a shard.core.Engine.
      • shard.plugins.serverside.DefaultGame - This is an off-the-shelf server plugin, running a default Shard game.
        • shard.plugins.serverside.Editor - This is the server-side Plugin for a Shard map editor.
      • shard.plugins.ui.UserInterface - This is the base class for an UserInterface for the Shard Client.
        • shard.plugins.pygameui.PygameUserInterface - This is a Pygame implementation of an UserInterface for the Shard Client.
          • shard.plugins.pygameui.PygameEditor - A Pygame-based user interface that serves as a map editor.
  • shard.FloorPlanElement - Convenience class for floor plan elements.
  • shard.interfaces.Interface - This is a base class for Shard interfaces which handle all the network traffic.
    • shard.interfaces.UDPClientInterface - This is the base class for a Shard Client Interface.
    • shard.interfaces.UDPServerInterface - This is the base class for a Shard Server Interface.
  • shard.interfaces.MessageBuffer - Buffer messages received and to be sent over the network.
    • shard.interfaces.UDPClientInterface - This is the base class for a Shard Client Interface.
  • shard.Message - A Message manages an ordered list of shard events.
  • shard.Rack - A Rack stores Entities removed from a room.
  • shard.run.App - An App instance represents a Shard client or server application.
  • shard.Tile - A Tile is an element of a two-dimensional map.
