Doctests for the Fabula Package
==============================

Asset Manager
-------------

    >>> stream = open("temp.dat", mode="w")
    >>> stream.write("fabula")
    6
    >>> stream.close()
    >>> import fabula.assets
    >>> import fabula.run
    >>> app = fabula.run.App("d", timeout = 5)
    >>> assets = fabula.assets.Assets(app.logger)
    >>> asset = assets.fetch("temp.dat")
    >>> asset.read()
    b'fabula'
    >>> asset.close()
    >>> import os
    >>> os.remove("temp.dat")
    >>>
